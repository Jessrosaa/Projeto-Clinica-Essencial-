# 🏥 Projeto Clínica Essencial

Aplicativo desenvolvido para solucionar problemas operacionais de uma clínica médica de pequeno porte, que enfrentava dificuldades na organização de agendamentos, controle de atendimentos e registro de informações clínicas.

## 🎯 Problema Identificado

A clínica apresentava:

- Falta de controle eficiente de agendamentos  
- Dificuldade para remarcação e cancelamento  
- Ausência de confirmação de consultas  
- Falta de registro estruturado dos atendimentos  
- Informações descentralizadas e não padronizadas

---

## 💡 Solução Desenvolvida

Criação de um aplicativo completo para centralizar:

- Página de agendamento para pacientes  
- Página de contato  
- Agenda interna para equipe  
- Prontuário eletrônico  
- Controle de status das consultas  

---

## 🧩 Funcionalidades do Sistema

### 📅 Gestão de Agendamentos
- Criar novo agendamento
- Editar consultas
- Remarcar atendimentos
- Cancelar consultas
- Alterar status (Confirmado, Cancelado, etc.)
- Confirmar consulta

### 🔎 Filtros Inteligentes
- Visualizar todos os agendamentos
- Filtrar agendamentos de **hoje**
- Filtrar agendamentos de **amanhã**

### 🗂️ Prontuário Eletrônico
- Registro de cada atendimento realizado
- Histórico de consultas
- Controle de tempo de retorno
- Centralização das informações clínicas
- Local para anexar exames e receitas

---

## 📊 Benefícios Gerados

- Centralização de dados
- Organização da agenda médica
- Redução de erros manuais
- Maior controle sobre atendimentos
- Padronização das informações clínicas
